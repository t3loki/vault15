04. Drupal Bootstrap sub-theming

  1. Check/Install JQuery Update module
  2. Get the base Bootstrap theme & put it inside yoursite/sites/all/themes/bootstrap
  3. Then copy the starter kit sub-theme into sites/all/themes and rename it to what you like. Then take that name and paste it as the name of the file bootstrap_subtheme.info.starterkit. So finally you will have my_custom_bootstrap_theme.info (without starterkit at the end)
  4. Ok, now open that my_custom_bootstrap_theme.info file in an editor. Write your name in the "name" field and in the description field.
  5. Get the latest Bootstrap source files. Put it inside your sub-theme folder. All the files should reside inside the folder Bootstrap, like your_sub_theme/Bootstrap/...files go here...
  6. Get the free Koala less compiler and add your custom sub-theme folder as a new project. Then find style.less -> click compile on the right side. The corresponding style.css is generated inside ./css of your sub-theme.
  7. The last step :)) As you use Bootstrap source files to run your theme you have to remove comments in the .info file, in the part that starts with METHOD 1: Bootstrap Source Files, so uncomment all the Bootstrap scripts in this part and check that the line settings[bootstrap_cdn] = '' is not commented, in this way you disable the Bootstrap CDN, that you don't use anyway.

02. Show/hide hidden files on Mac

For Mac OS X 10.8, 10.7, 10.6

defaults write com.apple.Finder AppleShowAllFiles TRUE
killall Finder

For OS X 10.9

defaults write com.apple.finder AppleShowAllFiles TRUE
killall Finder

01. Sublime Text. Features

Tab Triggered Auto-Complete

One of the greatest features of Sublime Text is it’s tab controlled auto-complete, customizable by snippets (more on snippets in the next step). To try it out, create a new file and type in “html” (without quotes) and with no trailing space, hit tab. You’ll see that it instantly sets you up a bare-bones html template, and places your cursor in between the title tags.

Multi-Select

Sublime Text 2 comes equipped with amazing multi-selection abilities. Consider a scenario in which you need to rename all occurrences of a word – perhaps you change the element class name in your CSS. Highlight a word, and hit CMD + CTRL + G. Voila – all occurrences of that word are now highlighted, and if you begin typing, you’ll see them all simultaneously being replaced. more to read

Disable auto tag completion

Go to Sublime Text -> Preferences -> Settings User -> add the line
"auto_close_tags": false
